スマブラインディアンポーカー

しょーぐん動画(https://www.youtube.com/watch?v=yaZzFtQIRFg)で紹介された
チーム乱闘をカードで決めたキャラとハンデ(%)で遊ぶゲーム

最大4人のプレーヤーに対して
- **スマブラテーマ**: キャラクターアイコンとハンデ値（%）を使用したポーカー。
- **リアルタイム対戦**: Firestore の `onSnapshot` を活用したシームレスな状態同期。
- **ゲストログイン**: ユーザー名入力のみで即座にプレイ可能（匿名認証）。
- **ホスト権限**:
  - ゲームの開始・中断（リセット）
  - ラウンド結果（優勝・最下位）の入力
  - 参加者（対戦者・観戦者）の強制退室（キック）機能 🆕
- **観戦者モード**:
  - 最大10名まで観戦者として参加可能。
  - ロビー内（待機画面）で、毎ゲームごとに「対戦に参加」か「観戦」かを自由に切り替えられます。
  - 観戦者は全員のカードを最初から見ることができます。
  - ホストは観戦者として進行に徹することも可能です。
- **レスポンシブデザイン**: モバイル・PCの両方に対応したプレミアムなUI。
キャラクタカード(86種類のスマブラキャラクタ計86枚)
ハンデカード(300%, 200%, 150%, 125%, 100%, 80%, 60%, 50%, 40%, 30%, 20%, 10%, 0%)それぞれ4枚ずつ計4*13枚
を1枚ずつ配り、自分に配られたカード以外が見える状態で、
大乱闘対戦をするか、降りるかを選択する
大乱闘を対戦して優勝した人に+1ポイント
最下位に-1ポイント
ゲームを降りた人、2位、3位は0ポイント
とする

システムとしては4人にログインしてもらい、専用部屋URLに入ってもらう
ゲームを開始するとカードをランダムに配る

配られたカードは各プレイヤーの画面に自分以外のカードを表示する

参加するか降りるかを決めたら（全員に意思表示の入力をさせる）全カードを表示する
対戦を実施し（ゲーム内）ゲームの結果優勝、最下位を入力させ、ポイント計算をする
次のカードを配る（キャラカードは一度配ったキャラは山に捨てる）

これを繰り返す
