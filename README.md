# ⚔️ スマブラ・インディアンポーカー

[![Netlify Status](https://api.netlify.com/api/v1/badges/03bdf940-ac44-43b1-8631-b860a15909aa/deploy-status)](https://app.netlify.com/projects/smash-indian-poker/deploys)

## 📌 これはなに？
[しょーぐん動画](https://www.youtube.com/watch?v=yaZzFtQIRFg)で紹介された、スマブラ・インディアンポーカーのカード配布部分をWebアプリ化しました。

自分以外のプレイヤーの「使用キャラクター」と「ハンデ（%）」だけが見える状態で、自分の強さを予想して対戦に参加するか、あるいは降りるかを決める「インディアンポーカー」形式のゲームです。

## 🚀 以下のリンクURLから遊ぶことができます
[https://smash-indian-poker.netlify.app/](https://smash-indian-poker.netlify.app/)

## 🎮 遊び方

### 1. 準備
- ホストが「新しいロビーを作る」を押し、発行された **ロビーID** を友だちに共有します。
- 参加者はそのロビーIDを入力して参加します（最大4名まで対戦可能、10名まで観戦可能）。

### 2. カード配布
- ゲームを開始すると、全員にランダムな「キャラクター」と「ハンデ%」が配られます。
- **自分のカードだけは見えません**。他人のカードをヒントに、自分の状況を推測します。

### 3. 駆け引きと決断
- 自分のカードが強い（＝低いハンデ%）と予想したら「参加」、負けそうなら「降りる」を選択します。
- 全員の意思が固まったら、ロビーIDの横にある「全公開」ボタンで一斉に公開します。

### 4. 大乱闘対戦
- 「参加」を選んだプレイヤー同士で、実際にスマブラの専用部屋を開き、配られた「キャラクター」と「ハンデ%」に設定して対戦を行います！

### 5. スコア記録
- 対戦後、ホストが結果（優勝・最下位）を入力して確定します。
  - 優勝者: **+1pt**
  - 最下位: **-1pt**
  - それ以外（2位/3位/降りた人）: **0pt**
- キャラクタカードは山札（全86種）から消費され、1ラウンド終了ごとに捨てられます。

## 🛠 技術スタック
- **Frontend**: Vite + Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS (Modern CSS variables, Flexbox/Grid, 3D Transforms)
- **Backend/DB**: Firebase Firestore (Real-time synchronization)
- **Auth**: Firebase Authentication (Anonymous Auth)
- **Hosting**: Netlify
- **Icons**: [スマブラ公式アセット](https://www.smashbros.com/)をベースにしたアイコンシステム

---

## ✨ 主な機能
- **リアルタイム同期**: 全プレイヤーの意思決定やカード公開が瞬時に反映されます。
- **リッチな演出**: 3Dフリップアニメーションによる臨場感のあるカード公開。
- **ホスト管理機能**: ルームの作成、キック（強制退出）、結果入力、ラウンドリセット。
- **観戦者モード**: 最大10名まで。全員のカードを見た状態でアドバイスや実況が可能です。
- **レスポンシブ**: PCはもちろん、スマホ片手にプレイしやすいUI設計。
